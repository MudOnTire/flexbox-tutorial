# 基本介绍

## 特点
1. flexbox是一种css display类型，提供一种更简单高效的布局方式；
2. flexbox可以对元素相对于父元素、兄弟元素进行定位、控制尺寸、控制间距；
3. flexbox对响应式有很好的支持；

## 工作原理
设置父元素的`display`属性为`flex`，则子元素都变成`flex item`，由此可以控制子元素的排列方式、尺寸、间距等；
